## Python

#### Recursion

> Recursion is the process where a function calls itself repeatedly
> 
> Let's take an example where a function uses recursion to find the nth power of a number

```
def power(x: int, n: int) -> int:
    if n == 1:
        return x
    return x * power(x, n - 1)
```
