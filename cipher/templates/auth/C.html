  About C

C is a high-level programming language developed at AT & T‚Äôs Bell Laboratories of USA in 1972. It was designed and written by Dennis Ritchie. It is reliable, simple and easy to use.

Features of C language:

  • Simplicity
  • Machine Independent or Portable

If you write a C code in your machine, it will run on any machine which supports C, without modifying a single line of code.

As it is not tied to any hardware or system, it is a hardware independent language or platform independent language.

That is why C is called a portable language.

  • Structured programming language

C is called a structured programming language because to solve a large problem, C programming language divides the problem into smaller structural blocks each of which handles a particular responsibility.

These structural blocks are ‚Äì

Decision making blocks like if-else-elseif, switch-cases, Repetitive blocks like for-loop, while-loop, do-while loop etc subroutines/procedures - functions

  • Rich Library
  • Memory Management
  • Fast Speed
  • Pointers

Pointers allows us to access the address of variables which is quite useful.

  • Extensible

C focuses on mechanisms to extend the programming language, compiler and runtime environment.

1Ô∏è‚É£ Description

Variables are also called identifiers. A variable is a quantity that can change its value during program execution.


For example, let us take a simple mathematical expression c = a + b. Here, the values of a, b, c are not known. To compute the value of c, different persons can give different values of a and b. As their values are not fixed, they are called variables.

2Ô∏è‚É£ Why variables?

In computers, data is stored in one or many locations in the primary memory. Each memory location has a unique address. To refer to this data, a name is to be assigned in a program. This name which is assigned to a particular memory location is called a variable. a variable is a container (storage area) to hold data. Variable names are just the symbolic representation of a memory location. These locations can contain integer, real or character constants.

‚û° The variable name must begin with either alphabet or an underscore(_) .

‚û° A maximum of 31 characters is allowed. But this is compiler dependent.

‚û° Keywords cannot be used as variable names.

‚û° No commas or blanks are allowed within a variable name.

‚û° The variable names are case sensitive.(Upper & Lower case alphabets are considered to be different, Ex: a-97, A-65).

4Ô∏è‚É£ The result of the conditional operator is the result of whichever operand is evaluated ‚Äî the second or the third. Only one of the last two operands is evaluated in a conditional expression.

The | (bitwise OR) in C or C++ takes two numbers as operands and does OR on every bit of two numbers. The result of OR is 1 if any of the two bits is 1.

The ›› (right shift) in C or C++ takes two numbers, right shifts the bits of the first operand, the second operand decides the number of places to shift.

The ~ (bitwise NOT) in C or C++ takes one number and inverts all bits of it.

Switch case statement:

A switch case statement is a selective control structure that selects a choice from the set of available choices.

It is very similar to if statement.

 But switch statement cannot replace if statement in all situations.

Syntax:

switch(Integer variable or Integer expression or character variable) {

    case Integer constant or character constant:

     statements;

     break;

    case Integer constant or character constant:

     statements;

     break;

    default: statements;

     break;

}

 Note: default can be written as first case or middle but good programming practice is to write it at the end.

The switch statement tests the value of variable or the expression with the set of constants. When a match is found, the statements associated with that constant are executed. When a match is not found the statements following default are executed but default is optional. The break statement is used to take the control out of the switch block. When there is no default and there are no matching constants ,then the program control goes out of the switch block.

A for loop is a repetition control structure that allows you to efficiently write a loop that needs to execute a specific number of times.

Syntax:

for (initialisation; condition; increment / decrement) {

    statement(s);

}

Here is the flow of control in a 'for' loop ‚àí

1Ô∏è‚É£ The init step is executed first, and only once. This step allows you to declare and initialize any loop control variables. You are not required to put a statement here, as long as a semicolon appears.

2Ô∏è‚É£ Next, the condition is evaluated. If it is true, the body of the loop is executed. If it is false, the body of the loop does not execute and the flow of control jumps to the next statement just after the 'for' loop.

3Ô∏è‚É£ After the body of the 'for' loop executes, the flow of control jumps back up to the increment statement. This statement allows you to update any loop control variables. This statement can be left blank, as long as a semicolon appears after the condition.

4Ô∏è‚É£ The condition is now evaluated again. If it is true, the loop executes and the process repeats itself (body of loop, then increment step, and then again condition). After the condition becomes false, the 'for' loop terminates.

JUMPING STATEMENTS IN LOOPS:

goto:

The goto statement allows us to transfer control of the program to the specified label.The label is an identifier. When the goto statement is encountered, the control of the program jumps to label: and starts executing the code.

             }

The use of goto statement may lead to code that is buggy and hard to follow. Also, the goto statement allows you to do bad stuff such as jump out of the scope. That being said, goto can be useful sometimes. For example: to break from nested loops.

break:

The break statement in C programming has the following two usages ‚àí

When a break statement is encountered inside a loop, the loop is immediately terminated and the program control resumes at the next statement following the loop.

It can be used to terminate a case in the switch statement (covered in the next chapter).

If you are using nested loops, the break statement will stop the execution of the innermost loop and start executing the next line of code after the block.

Syntax

The syntax for a continue statement in C is as follows ‚àí

break;

The continue statement in C programming works somewhat like the break statement. Instead of forcing termination, it forces the next iteration of the loop to take place, skipping any code in between.

For the for loop, continue statement causes the conditional test and increment portions of the loop to execute. For the while and do...while loops, continue statement causes the program control to pass to the conditional tests.

Syntax

The syntax for a continue statement in C is as follows ‚àí

continue;

  Functions

A function is a self-contained block of statements that perform a coherent task of some kind. Every C program can be thought of as a collection of these functions.

‚û° Any C program contains at least one function which must be main().

‚û° Program execution always begins with main( ).

‚û° There is no limit on the number of functions that might be present in a C program.

‚û° After each function has done its thing, control returns to main( ). When main( ) runs out of function calls, the program ends.

‚û° C program is a collection of one or more functions. A function gets called when the function name is followed by a semicolon.

‚û° A function is defined when function name is followed by a pair of braces in which one or more statements may be present.

‚û° Any function can be called from any other function. Even main( ) can be called from other functions.

‚û° A function can be called any number of times.

‚û° The order in which the functions are defined in a program and the order in which they get called need not necessarily be same. However, it is advisable to define the functions in the same order in which they are called. This makes the program easier to understand.

Advantages of using functions in C

‚Ä¢ The program can be modularised into smaller parts using functions.

‚Ä¢ The function provides reusable code.

‚Ä¢ In large programs, debugging and editing tasks is easy with the use of functions.

‚Ä¢ Separate function independently can be developed according to the needs.

Parts of Function

1. Function Prototype (function declaration)

2. Function Definition

3. Function Call

o Function declaration: A function must be declared globally in a c program to tell the compiler about the function name, function parameters, and return type.

o Function call: Function can be called from anywhere in the program. The parameter list must not differ in function calling and function declaration. We must pass the same number of functions as it is declared in the function declaration.

o Function definition: It contains the actual statements which are to be executed. It is the most important aspect to which the control comes when the function is called. Here, we must notice that only one value can be returned from the function.

Call by value:

‚Ä¢ The actual parameter is passed to a function.

‚Ä¢ New memory area created for the passed parameters, can be used only within the function.

‚Ä¢ Instead of copying variable; an address is passed to function as parameters.

‚Ä¢ Changes in function reflect the change of the original variables.

C variable scope:

A scope is a region of the program, and the scope of variables refers to the area of the program where the variables can be accessed after its declaration. In C programming, variable declared within a function is different from a variable declared outside of a function.

The variable can be declared in three places.They are -

Inside a function or a block ‚Äì local variables

In the function parameters - formal parameters

Out of all function - global variables

Local (private) variables: Variables declared inside a function block are called local to that function and the scope of those variables is limited to that function.

Global (public) variables: Variables declared outside of a function block are called global and are available for all functions from the declared position outwards.

 Note: If global and local parameters fall on same name then preference is given to local only.

Arrays

Arrays are referred to as structured data types. An array is defined as a finite ordered collection of homogeneous data, stored in contiguous memory locations.

1Ô∏è‚É£ Single Dimensional Array :

‚Ä¢ Declaring an array:

Like any other variable, arrays must be declared before they are used. General form of array is ,

data_type variable_name[size];

ex: int arr[10];

Here int is the data type, arr is the name of the array and 10 is the size of the array. It means array arr can only contain 10 elements of int type.

‚Ä¢ Index of an array starts from 0 to size-1 i.e first element of arr array will be stored at arr[0] address and the last element will occupy arr[9].

Initialisation of array:

‚Ä¢ After an array is declared it must be initialised. Otherwise, it will contain garbage value (any random value).An array can be initialised at either compile time or at runtime.

Here are a few examples,

int marks[4]={1,2,3,4};

float area[5]={23.4,6.2,5.5};

‚Ä¢ An array can also be initialised at runtime using scanf() function. This approach is usually used for initialising large arrays, or to initialise arrays with user specified values.

2Ô∏è‚É£ Two dimensional array:

‚Ä¢ C language supports multidimensional arrays also. The simplest form of multidimensional array is a two dimensional array. Both the rows and columns index begins from 0.

‚Ä¢ Two dimensional array are declared as follows:

data_type array_name[row_size][column_size];

Ex: int arr[3][4];

‚Ä¢ An array can also be declared and initialized together. For example,

int arr[][4]={{0,0,0}, {1,1,1||;

 Note: We have not assigned any row value to our array in the above example. It means we can initialize any number of rows. But, we must always specify the number of columns, else it will give a compile time error. Here, a 2*3 multi-dimensional matrix is created.
